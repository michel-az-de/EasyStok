   c/ 5 commits TASK-EZ-AVAL-001 ahead — trabalho de outra sessao, R6).

   ocupar a maquina local (que tinha o dotnet test rodando em paralelo).

   Docker. Suite restante cobre 1925 testes — abrange Architecture, Domain,
   Application, Api Unit, Web Unit. Suficiente para validar deploy.

   exit silencioso 0. Trocado para rodar projeto-a-projeto sem essa flag,
   adicionando ~3min ao tempo total. Vale a pena pelo determinismo.

5. **fly deploy SEQUENCIAL** (api → admin → web), nao paralelo. Razao:
   admin e web dependem da api respondendo (consumem ApiBaseUrl). Se algo
   quebrar, falha cedo. Total ~10min.

   "smoke HTTP estrutural — eu rodo" no inicio. Cobre os 7 modulos via
